Подробнее о системе

Система персонализации контента позволяет в реальном времени создавать различные подборки контента в рамках пути пользователя, тем самым создавая персональный путь для каждого пользователя.

Основные метрики:

- Уменьшение времени между заказами / транзакций

- Уменьшение времени до совершения самого заказа

- Увеличение объема заказа

Сопутствующие метрики:

- Увеличение частоты заказов / транзакций

- Увеличения объема совокупного заказа / транзакции

- Создание положительного опыта взаимодействия с брендом
